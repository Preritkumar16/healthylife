# HealthyLife - Your Wellness Companion 🚀

[![Vite](https://img.shields.io/badge/Vite-6.2.0-green.svg)](https://vitejs.dev/) [![React](https://img.shields.io/badge/React-19-blue.svg)](https://reactjs.org/) [![TypeScript](https://img.shields.io/badge/TypeScript-5.8-blue.svg)](https://www.typescriptlang.org/)

HealthyLife is a modern, responsive web app for personalized health & wellness. Features BMI calculator, diet plans, workouts, mindfulness exercises, and home dashboard.

## ✨ Features
- 📱 Fully responsive design (mobile-first)
- 🎨 Smooth animations with Framer Motion
- 🗺️ Client-side routing with React Router
- 📊 Interactive BMI calculator
- 💪 Workout & diet recommendations
- 🧘 Guided mindfulness sessions
- 🎨 Tailwind CSS styling

## 🛠 Tech Stack
- React 19 + TypeScript
- Vite (build tool)
- Tailwind CSS 4
- Framer Motion (animations)
- Lucide React (icons)
- React Router DOM

## 🚀 Quick Start

1. **Clone & Install**
   ```bash
   npm install
   ```

2. **Run Development Server**
   ```bash
   npm run dev
   ```
   App runs at [http://localhost:3000]

3. **Build for Production**
   ```bash
   npm run build
   ```
